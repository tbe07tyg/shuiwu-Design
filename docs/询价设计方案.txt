# 项目调研模块 - 询价流程设计方案

## 1️⃣ 询价流程业务梳理

```
┌─────────────────────────────────────────────────────┐
│  🔔 询价流程（供应商报价与方案收集）                │
├─────────────────────────────────────────────────────┤
│  1. 需求方在系统内发起询价，选择/导入供应商         │
│  2. 系统向供应商库手机号发送唯一链接                │
│  3. 供应商通过链接填写报价、上传方案PPT             │
│  4. 所有方案自动归档到方案库，按需求类别分类        │
│  5. 方案支持后续反复查阅、下载、归档                │
└─────────────────────────────────────────────────────┘
```

---

## 2️⃣ 主要页面与功能

### 2.1 询价发起页面（系统内）

```
┌─────────────────────────────────────────────────────┐
│  📋 询价发起                                        │
│  ┌───────────────────────────────────────────────┐ │
│  │ 需求选择： [多选需求]                          │ │
│  │ 供应商选择： [多选]                            │ │
│  │ 发送方式： [📤 发送短信] [🔗 复制链接]          │ │
│  └───────────────────────────────────────────────┘ │
│  ┌───────────────────────────────────────────────┐ │
│  │ 发送历史                                      │ │
│  │ 供应商 | 需求 | 发送方式 | 链接 | 发送时间 | 回执时间 | 状态 | 操作 │ │
│  │ 张三   | 智能水表 | 短信 | ... | 2024-06-01 10:00 | —        | 已发送       | 短信提醒     │ │
│  │ 李四   | 智能水表 | 复制 | ... | 2024-06-01 10:01 | 2024-06-01 11:20 | 已提交回执   | 查看回执/短信提醒 │ │
│  │ ...                                         │ │
│  └───────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

#### 交互说明
- 链接栏右侧有复制按钮，操作栏不再重复"复制"。
- 操作栏：
  - 有回执时显示"查看回执"和"短信提醒"。
  - 无回执时只显示"短信提醒"。
- 状态栏支持"已发送""已复制""已提交回执"。
- 发送时间、回执时间分别记录操作和供应商提交时间。

## 3️⃣ 数据流与安全

- 供应商提交页面需校验token，防止伪造和重复提交。
- 方案文件按需求类别、项目、供应商多维度归档。
- 方案库支持权限管理、历史版本、全文检索。

---

## 4️⃣ 交互与UI风格

- 国企风格，稳重、简洁、分区明确。
- 重要操作按钮突出，表格/卡片式布局，文件上传区美观易用。
- 外部页面适配手机端，操作流程简洁明了。

---

## 5️⃣ 可扩展点

- 支持批量导入供应商、批量发送询价。
- 供应商提交后自动短信/微信通知项目负责人。
- 方案库支持多维度统计分析、归档、备注。
