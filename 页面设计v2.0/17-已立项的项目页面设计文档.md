# ✅ 已立项的项目页面设计文档 v2.0

## 📄 页面基础信息

- **页面名称**: 已立项的项目页面
- **路由地址**: `/approval/review`
- **组件路径**: `src/views/approval/review/index.vue`
- **设计版本**: v2.0
- **更新时间**: 2025-01-17
- **实现状态**: 📋 设计阶段

## 🎯 功能定位

已立项的项目页面是立项后的项目管理入口，提供已通过立项审批的项目总览、状态跟踪、启动管理和转入实施的功能，实现从立项到实施的无缝衔接。

## ⚡ 交互逻辑

### 📋 项目加载
```javascript
/**
 * 加载已立项项目列表
 */
const loadApprovedProjects = async () => {
  const result = await fetchApprovedProjects({
    status: filterStatus.value,
    keyword: searchKeyword.value,
    page: currentPage.value
  });
  
  if (result.success) {
    projectList.value = result.data.projects;
    updateStatistics(result.data.statistics);
  }
};
```

### 🚀 项目启动
```javascript
/**
 * 启动项目
 */
const startProject = async (projectId, startupData) => {
  // 验证启动条件
  const checkResult = await validateStartupConditions(projectId);
  if (!checkResult.valid) {
    showError(checkResult.message);
    return;
  }
  
  // 创建项目实例
  const result = await createProjectInstance({
    projectId,
    startDate: startupData.startDate,
    kickoffMeeting: startupData.kickoffMeeting,
    projectCharter: startupData.projectCharter
  });
  
  if (result.success) {
    // 更新项目状态为实施中
    await updateProjectStatus(projectId, 'in_progress');
    // 生成项目管理看板
    await createProjectDashboard(result.data.instanceId);
    // 发送启动通知
    await sendStartupNotifications(projectId);
  }
};
```

### 📊 报表生成
```javascript
/**
 * 生成项目报表
 */
const generateProjectReport = async (projectIds, reportType) => {
  const result = await generateReport({
    type: reportType, // 'summary', 'budget', 'timeline'
    projectIds: projectIds,
    format: 'excel' // 'excel', 'pdf', 'word'
  });
  
  if (result.success) {
    // 下载报表文件
    downloadFile(result.data.downloadUrl, result.data.filename);
  }
};
```
