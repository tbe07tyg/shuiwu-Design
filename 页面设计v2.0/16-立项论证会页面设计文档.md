# 👥 立项论证会页面设计文档 v2.0

## 📄 页面基础信息

- **页面名称**: 立项论证会页面
- **路由地址**: `/approval/decision`
- **组件路径**: `src/views/approval/decision/index.vue`
- **设计版本**: v2.0
- **更新时间**: 2025-01-17
- **实现状态**: 📋 设计阶段

## 🎯 功能定位

立项论证会页面是立项决策的关键环节，提供论证会议的组织、专家邀请、会议记录、决策投票和结果公示等功能，确保立项决策的科学性和公正性。

## 📐 设计规格

### 🌈 色彩方案
- **页面背景**: `#f5f8ff` (淡蓝色)
- **卡片背景**: `#ffffff` (纯白)
- **边框颜色**: `#e8e8e8` (浅灰)
- **状态颜色**:
  - 待开会: `#faad14` (橙色)
  - 进行中: `#1890ff` (蓝色)
  - 已完成: `#52c41a` (绿色)
- **投票颜色**:
  - 同意: `#52c41a` (绿色)
  - 有条件同意: `#faad14` (橙色)
  - 不同意: `#ff4d4f` (红色)
- **主色调**: `#234fa2` (深蓝色)

### 📏 尺寸规格
- **卡片间距**: `16px`
- **按钮高度**: `32px`
- **表格行高**: `48px`
- **进度条高度**: `8px`
- **圆角**: `6px`

### 📝 字体规格
- **页面标题**: 24px, font-weight: bold
- **卡片标题**: 18px, font-weight: 600
- **正文内容**: 14px
- **投票结果**: 16px, font-weight: 500

## ⚡ 交互逻辑

### 📅 会议管理
```javascript
/**
 * 创建论证会
 */
const createMeeting = async (meetingData) => {
  const result = await createEvaluationMeeting({
    projectId: meetingData.projectId,
    scheduleTime: meetingData.scheduleTime,
    location: meetingData.location,
    chairperson: meetingData.chairperson,
    experts: meetingData.experts
  });
  
  if (result.success) {
    // 发送邀请通知
    await sendInvitations(meetingData.experts);
    // 更新列表
    await loadMeetingList();
  }
};
```

### 🗳️ 投票系统
```javascript
/**
 * 专家投票
 */
const submitVote = async (voteData) => {
  const result = await submitExpertVote({
    meetingId: currentMeeting.value.id,
    expertId: userInfo.value.id,
    decision: voteData.decision, // 'approve', 'conditional', 'reject'
    comment: voteData.comment,
    scores: voteData.scores
  });
  
  if (result.success) {
    // 更新投票统计
    await updateVoteStatistics();
    // 检查是否投票完成
    await checkVotingCompletion();
  }
};
```

### 📝 会议记录
```javascript
/**
 * 实时保存会议记录
 */
const autoSaveMeetingRecord = debounce(async (content) => {
  await saveMeetingRecord({
    meetingId: currentMeeting.value.id,
    content: content,
    timestamp: new Date()
  });
}, 2000);
```

## 🎯 用户体验亮点

### ⚡ 智能化功能
1. **自动提醒**: 会议前自动发送提醒通知
2. **智能匹配**: 根据项目类型推荐合适专家
3. **进度跟踪**: 实时跟踪会议和投票进度

### 🔄 协作功能
1. **多端同步**: 支持多设备同步会议状态
2. **实时互动**: 专家可实时查看讨论和投票
3. **移动支持**: 专家可通过手机参与投票

### 📊 数据洞察
1. **投票分析**: 详细的投票统计和趋势分析
2. **专家评估**: 专家参与度和评议质量分析
3. **决策追踪**: 完整的决策过程记录
