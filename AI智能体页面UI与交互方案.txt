# AI智能体页面UI与交互方案

## 1. 典型页面UI草图

### 需求池管理
```
┌──────────────────────────────────────────────┐
│  需求池管理                                  │
│  [新建需求]  [AI智能优化]                    │
├──────────────────────────────────────────────┤
│  需求列表                                    │
│  ┌────────────────────────────────────────┐ │
│  │ 需求标题   标签   优先级   状态         │ │
│  │ 智能水表   [AI推荐:技术] 高    进行中   │ │
│  │ [AI摘要]  [AI合并建议]                 │ │
│  └────────────────────────────────────────┘ │
│  右侧AI助手：                               │
│  Q: 帮我总结本需求？                        │
│  Q: 推荐相关案例？                          │
└──────────────────────────────────────────────┘
```

### 材料上传/编辑
```
┌──────────────────────────────────────────────┐
│  材料上传/编辑                               │
│  [上传文件]  [AI一键补全]  [AI自动审核]      │
├──────────────────────────────────────────────┤
│  材料内容区                                 │
│  ┌────────────────────────────────────────┐ │
│  │ ...                                   │ │
│  └────────────────────────────────────────┘ │
│  AI审核弹窗：                              │
│  - 缺少预算说明                            │
│  - 格式不符                                │
│  [一键采纳] [人工编辑]                     │
└──────────────────────────────────────────────┘
```

### 进度监控
```
┌──────────────────────────────────────────────┐
│  进度监控                                   │
│  [AI风险预警]  [AI生成报告]                  │
├──────────────────────────────────────────────┤
│  项目进度图表                               │
│  AI助手：                                   │
│  - 本月有2个节点进度滞后，建议调整资源       │
│  - 生成进度趋势分析报告                     │
└──────────────────────────────────────────────┘
```

## 2. AI交互流程图（Mermaid语法）

```mermaid
flowchart TD
    A[用户进入页面] --> B{页面类型}
    B -- 需求池管理 --> C1[AI标签/优先级推荐卡片]
    C1 --> D1[用户采纳/编辑建议]
    C1 --> E1[AI一键合并/批量操作按钮]
    B -- 材料上传/编辑 --> C2[AI自动审核弹窗]
    C2 --> D2[审核意见展示/一键采纳]
    C2 --> E2[AI补全/优化按钮]
    B -- 进度/节点/付款/合同 --> C3[AI风险预警/趋势分析按钮]
    C3 --> D3[AI生成报告/异常提醒]
    B -- 统计分析 --> C4[AI一键分析/智能问答入口]
    C4 --> D4[AI生成可视化报告/决策摘要]
    B -- 日常维护/知识库 --> C5[AI知识问答/推荐卡片]
    C5 --> D5[用户提问/一键操作]
    B -- 全局 --> C6[悬浮AI助手/主动推送]
    C6 --> D6[对话式交互/批量操作/提醒]
```
